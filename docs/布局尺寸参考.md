│  │         (afterTitle 14px)         │           │
│  ├─── 252 ───┬── 10 ──┬─── 402 ───┤           │
│  │ 本命法器   │  gap   │  角色形象  │           │
│  │ 252 × 540 │        │ 402 × 540  │  mainH   │
│  │            │        │            │  =540    │
│  ├────────────┴────────┴────────────┤           │
│  │         (sectionGap 16px)         │           │
│  ├──── 327 ────┬─ 10 ─┬──── 327 ───┤           │
│  │  基础属性    │ gap  │  功法造诣   │  statsH  │
│  │  327 × 290  │      │  327 × 290  │  =290   │

│ PAD=16                                    PAD=16│

| 子面板 | 宽度 | 高度 | 比例 |
