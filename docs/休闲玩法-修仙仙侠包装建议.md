# 休闲小游戏 → 修仙/仙侠包装建议

把现有「下100层 + 抽奖转盘 + 洞府养成」包装成修仙/仙侠题材，从**叙事、用词、表现、小系统**四块入手，不改核心循环也能明显提升「修仙感」。

---

## 一、叙事包装（世界观与动机）

| 当前 | 修仙化包装 | 说明 |
|------|------------|------|
| 进入秘境 | **闯秘境 / 入洞天 / 探灵脉** | 强调「机缘与凶险并存」的未知感 |
| 每层开格 | **破禁制 / 探灵机 / 择机缘** | 开格 = 破开一层禁制，露出灵机/陷阱/妖兽 |
| 下一层 | **深入一层 / 再下一重** | 文案可带「越深机缘越大，凶险越重」 |
| 撤离 | **收功出关 / 见好就收 / 暂避锋芒** | 撤出 = 主动收功，不是「逃跑」 |
| 死亡 | **神识受损 / 道基动摇 / 被迫出关** | 弱化挫败感，强调「下次再探」 |
| 抽奖转盘 | **天机莫测 / 祸福相依 / 因果轮盘** | 已有「祸兮福所倚、福兮祸所伏」，可加一句「天机一线，各凭缘法」 |

**一句话锚定**：  
「你在洞府修行，为求突破需入秘境历练——每层破禁探机缘、斗妖兽、遇祸福转盘，见好就收或冲到底，全在一念之间。」

## 二、用词与命名（全链路修仙化）

### 1. 境界与成长

| 当前 | 建议 | 备注 |
|------|------|------|
| 境界 1/2/3 层 | **炼气一重 / 炼气二重 / 炼气三重**，再往后 **筑基 / 金丹** | 文档里已提，可直接替换数字为境界名 |
| 修炼突破 | **突破境界 / 破境 / 渡小劫** | 按钮与提示统一 |
| 修为 | 保持「修为」或改为 **道行 / 灵力** | 与「法力」区分：修为=长期，法力=当局 |

| 开始抽奖 | **窥天机 / 转因果 / 一试缘法** |

## 三、表现力（视听与氛围）

在不改玩法的前提下，用**文案 + 简单演出**强化修仙感：

1. **洞府界面**
   - 标题：**「洞府」 / 「修行静室」**，副标题保留「是仙人就下100层」或改为「秘境历练」。
   - 突破：点击「修炼突破」后，简短 **突破演出**（光效/闪白 + 一字提示如「破境」「筑基」），再刷新数值。

2. **秘境界面**
   - 顶部：**「第 N 重」** 替代「第 N 层」，或「秘境·第 N 重」。
   - 生命/法力：可写 **「气血」「灵力」**，或保留「生命」「法力」但用略古风的颜色/字体。

3. **战斗**
   - 进入战斗：一句 **「妖兽现身！」/「煞气凝形！」** 再切战斗。
   - 符咒：给技能起名（如 **「掌心雷」**），释放时短文案 **「符箓·雷」** 或范围圈/轨迹。

4. **转盘**
   - 标题已用「祸兮福所倚」「福兮祸所伏」，可加副标题 **「天机莫测，各凭缘法」**。
   - 按钮「开始抽奖」→ **「窥天机」/「转因果」**。

5. **结算**
   - 撤离：**「收功出关，携灵石 XXX、灵药 XXX 返回洞府」**。
   - 死亡：**「神识受损，道基动摇，被迫出关」**。
   - 通关百层：**「百重秘境，一朝勘破」** 或类似。

## 四、可做的小系统（轻量修仙化）

在不破坏「短局、休闲」的前提下，可择一两个加入：

| 系统 | 做法 | 修仙感 |
|------|------|--------|
| **境界命名表** | 1–10 炼气一重～十重，11–20 筑基初～圆满，21+ 金丹等 | 成长有「修仙阶段」 |
| **突破演出** | 突破时 0.5s 光效 + 一字「破境」「筑基」 | 反馈清晰、有仪式感 |
| **符咒命名/多种** | 当前符咒命名「掌心雷」，后续可多做一个「离火符」等选带 | 战斗更有辨识度 |
| **层数主题/别名** | 如第 7 层「灵药层」、第 13 层「凶险层」，仅文案/权重 | 每层有记忆点 |
| **心魔/天劫（可选）** | 每 N 层或抽到某减益时，弹一句「心魔扰动」再结算效果 | 祸福更有叙事 |

## 五、实施优先级建议

| 优先级 | 内容 | 实现量 | 效果 |
|--------|------|--------|------|
| P0 | 全文案替换：境界名（炼气/筑基）、生命→气血、法力→灵力、转盘按钮「窥天机」等 | 小 | 整体修仙味立刻上来 |
| P0 | 突破时简短光效 + 一字「破境」 | 小 | 养成有仪式感 |
| P1 | 战斗进场一句「妖兽现身！」、符咒命名「掌心雷」 | 小 | 战斗更贴题材 |
| P1 | 结算/撤离/死亡文案统一成「收功出关」「神识受损」等 | 小 | 叙事闭环 |
| P2 | 境界表扩展（炼气十重→筑基→金丹）、层数主题别名 | 中 | 深度与差异感 |
| P2 | UI 主色/形状略向墨、符、灵气感靠拢 | 中 | 氛围加分 |

---

## 六、总结

- **休闲玩法不变**：仍是「开格 → 战斗/资源/转盘 → 下一层/撤出」，短局、可重复。
- **包装做足**：叙事一句锚定、全链路用词修仙化、关键节点加一句台词或短演出。
- **优先做**：文案替换 + 突破演出 + 战斗/转盘/结算的修仙文案，即可把「休闲小游戏」包装成「修仙/仙侠历练」感。

这样既保留休闲节奏，又让玩家一眼觉得「这是修仙/仙侠游戏」。
