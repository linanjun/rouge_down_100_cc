# 功法分神念 · 多秘境世界设计

把「下100层」包装成：**修炼功法 → 分出神念 → 神念幻化进入不同秘境世界**；每个秘境有独立故事线与差异化产出。

---

## 一、顶层叙事

- **本体**：在洞府中修炼功法、积累修为与资源。
- **分神念**：功法到一定境界可「分出一缕神念」，神念不伤本体，可入幻境历练。
- **秘境世界**：神念可选择的「幻化世界」——每个世界是一段独立的故事与规则，历练后神念回归，把**该世界的产出**带回洞府（转化为修为/灵石/专属资源）。

**一句话**：  
「你修的是可分化神念的功法；神念投入不同秘境世界历练，每个世界故事不同、产出不同，最后都反哺本体修行。」

---

## 二、秘境 = 故事线 + 命名主题 + 产出

每个「秘境」在结构上统一为「下 N 层 + 开格 + 战斗 + 转盘 + 撤离」，但有三层包装可区分：

| 维度 | 说明 | 示例 |
|------|------|------|
| **故事/背景** | 该秘境是谁的幻境、什么主题、一句话梗概 | 凡人修仙传：韩立记忆中的险恶修仙界 |
| **命名主题** | 格子/敌人/陷阱/资源在本世界的叫法 | 灵植→灵药，妖兽→魔修；或 灵植→零件，妖兽→机械兽 |
| **产出** | 带出的资源类型与用途 | 灵石+灵药+天材地宝；或 灵石+搞笑币+吐槽结晶；或 能量+图纸+机械核心 |

下面给三个示例秘境，可直接当模板用或再衍生。

---

## 三、示例秘境一：凡人修仙传秘境

- **故事线**：神念落入「某位凡人修士记忆中的修仙界」——资源匮乏、步步凶险、机缘与杀机并存。
- **氛围**：谨慎、苟道、见好就收。

| 元素 | 命名/文案 |
|------|-----------|
| 灵植 | 灵药、黄龙丹 |
| 灵石 | 灵石、低阶灵石 |
| 天材地宝 | 天材地宝、奇物 |
| 妖兽/Boss | 妖兽、魔修、守关妖 |
| 陷阱 | 禁制、煞气、幻阵 |
| 机缘 | 机缘、福缘 |
| 层数 | 第 N 重 / 深渊第 N 层 |
| 撤离 | 神念收归，携收获返回 |

**产出（带回洞府）**  
- 灵石（通用）  
- 灵药（可换算修为或专用）  
- 天材地宝（稀有，可用于突破或兑换）  
- 修为（历练所得，与现有修为统一）

**故事句**（选一句做秘境入口/说明）：  
「神念所至，乃是凡人修仙界一隅——机缘与杀机同在，谨慎方能走远。」

---

## 四、示例秘境二：没钱修什么仙秘境

- **故事线**：神念落入「没钱修什么仙」的荒诞世界——修炼要钱、机缘要钱、处处要钱，又穷又搞笑。
- **氛围**：轻松、吐槽、穷开心。

| 元素 | 命名/文案 |
|------|-----------|
| 灵植 | 路边灵草、便宜丹药 |
| 灵石 | 灵石、铜板、打工钱 |
| 天材地宝 | 捡漏奇物、地摊宝贝 |
| 妖兽/Boss | 讨债的、拦路怪、黑心掌柜 |
| 陷阱 | 账单、罚款、套路贷 |
| 机缘 | 意外之财、白嫖机缘 |
| 层数 | 第 N 关 / 穷途第 N 站 |
| 撤离 | 揣着家当赶紧撤 |

**产出（带回洞府）**  
- 灵石（通用）  
- **搞笑币/吐槽结晶**（本秘境专属，可用于兑换皮肤、称号或搞笑向加成）  
- 灵药/天材地宝（可弱化或改为「打工所得」）  
- 修为（略少，强调「穷修」）

**故事句**：  
「神念一入此界，满眼都是穷——机缘要钱、破阵要钱，唯有见好就收，才能揣着家当回来。」

---

## 五、示例秘境三：超神机械师秘境

- **故事线**：神念落入「机械师」所在的星际/机械世界——敌人是机械与改造体，资源是能量与图纸。
- **氛围**：科幻、机械、升级感。

| 元素 | 命名/文案 |
|------|-----------|
| 灵植 | 能量块、补给包 |
| 灵石 | 能源、星币 |
| 天材地宝 | 稀有图纸、核心部件 |
| 妖兽/Boss | 机械兽、改造体、守关机甲 |
| 陷阱 | 电网、自爆、系统入侵 |
| 机缘 | 数据包、升级协议 |
| 层数 | 第 N 区 / 深渊第 N 层 |
| 撤离 | 神念回传，携带数据与物资返回 |

**产出（带回洞府）**  
- **能源/星币**（可 1:1 或比例换算为洞府「灵石」）  
- **图纸/机械核心**（本秘境专属，可用于解锁机甲皮肤、机械风符咒表现或被动）  
- 灵药可改为「修复包」「纳米药剂」→ 仍换算修为或生命相关  
- 修为（历练经验，可统一或略少）

**故事句**：  
「神念接入异界网络——此地无灵气，唯有能源与机械；带回图纸与核心，或可反哺本体修行。」

## 六、统一玩法结构（便于实现）

所有秘境共用同一套玩法逻辑，用「配置表」区分故事与产出：

```
秘境配置 = {
  id: string;           // 如 "fanren" | "meiqian" | "chaoshen"
  name: string;         // 显示名：凡人修仙传秘境 / 没钱修什么仙秘境 / 超神机械师秘境
  storyShort: string;   // 入口/说明用一句话
  naming: {             // 文案替换（可选，未填则用默认修仙）
    herb?: string;      // 灵植→灵药 / 能量块
    stone?: string;     // 灵石→灵石 / 星币
    treasure?: string;  // 天材地宝→奇物 / 图纸
    monster?: string;   // 妖兽→魔修 / 机械兽
    trap?: string;      // 陷阱→禁制 / 电网
    buff?: string;      // 机缘→福缘 / 数据包
    boss?: string;      // Boss→守关妖 / 守关机甲
    layer?: string;     // 第N层→第N重 / 第N区
    withdraw?: string;  // 撤离→神念收归 / 揣着家当撤
  };
  rewards: {            // 本秘境产出在洞府的用途/名称
    spirit: string;     // 灵石 / 星币
    herb: string;       // 灵药 / 搞笑币 / 补给包
    treasure: string;   // 天材地宝 / 吐槽结晶 / 图纸
  };
}
```

- **当前项目**：可先把现有「下100层」当作**默认秘境**（如「凡人修仙传秘境」），再增加「秘境选择」入口。
- **选择流程**：洞府界面「分神念入秘境」→ 弹出秘境列表（名称 + 一句故事）→ 选一个 → 进入该秘境的「下100层」玩法，文案与产出按配置切换。

## 七、实现步骤建议

| 步骤 | 内容 | 说明 |
|------|------|------|
| 1 | 叙事与入口文案 | 洞府「进入秘境」改为「分神念入秘境」；加载时一句「神念幻化，可择一秘境历练」 |
| 2 | 单秘境配置化 | 把当前格子类型、战斗、转盘、结算的**文案**抽成一份「默认秘境配置」，逻辑不变 |
| 3 | 秘境选择界面 | 一个简单列表：秘境名 + 一句 storyShort，点击后 setCurrentRealm(id) 再 startExpedition() |
| 4 | 多秘境配置表 | 新增凡人修仙传 / 没钱修什么仙 / 超神机械师 三份配置，naming + rewards 不同 |
| 5 | 产出差异化 | 根据 currentRealm 决定：结算时显示的奖励名称、以及是否写入「专属货币」（如搞笑币、图纸数量） |
| 6 | 后续扩展 | 专属货币可兑换皮肤、称号、或解锁新秘境入口 |

这样，**同一套「下100层 + 转盘 + 战斗」**就能包装成多个「故事不同、产出不同」的秘境世界，而本体始终是「修炼功法、分神念、神念入秘境历练再回归」。
