## 1. 统一风格结论

这份文档中的所有提示词，统一采用以下方向：

- 深色像素风修仙 UI
- 深蓝灰石板底
- 铜金包边与机关角饰
- 低饱和高对比
- 小尺寸高识别度图标
- 复古手游功能页风格
- 东方修真题材，但不是写意卡绘
- 图标默认要能嵌进深色面板里使用

- 轻水墨
- 写意卡绘小图标
- 玉石流光主导的厚涂 icon

## 4. 通用出图约束

- 单图只保留一个主物件
- 主体占画面 70% 到 85%
- 默认透明背景
- 适合 64x64 / 96x96 / 128x128 像素 icon
- 优先正视角或轻微三分之二视角
- 轮廓必须一眼可识别
- 少量高光即可，不要大面积雾效
- 高光服务功能识别，不追求仙气大场景
- 默认考虑嵌入深色石板铜框面板后的可读性

## 4.1 四类专用模板

以下四类模板用于解决同一题材在不同 UI 区域里显示尺度不同的问题。不要把详情图标直接缩小去当底栏 icon，也不要把底栏小 icon 放大后拿去做详情主图。

适用位置：底部主导航、一级页签、小型栏目切换

- 64x64
- 72x72
- 80x80

表现要求：

- 极简轮廓
- 单主体
- 少细节
- 强辨识

## 4.2 四类模板的使用规则

- 底栏页签：优先符号化，少细节，强读形
- 快捷入口：优先功能感，允许少量质感
- 列表小图标：优先类别区分，细节中等
- 详情大图标：优先主体完整度，但仍保持像素风和透明背景
- 同一题材跨四类使用时，主体形状要保持一致，只调整细节层级

## 11. 批量生产规则

- 功法统一使用“秘籍封面 / 玉简本体”结构
- 灵宠统一使用“头像 / 半身像素灵兽”结构
- 法器统一使用“单件器物”结构
- 丹药统一使用“单枚丹丸 / 小型丹瓶”结构
- 商城道具统一使用“袋、匣、令牌、玉简、箱盒”结构
- 同一批次固定母提示词，不要混入写意卡绘风
- 稀有度差异优先通过边色、内发光、角饰等级表达
- 所有图标都要默认可嵌入深色石板铜框 UI 面板
- 如果用于列表小图标，优先保轮廓而不是保材质细节
