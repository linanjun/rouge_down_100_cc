# 陷阱与机缘 · 小玩法建议

在现有「开格→陷阱扣血 / 机缘加攻」基础上，用**子类型**或**一次性格子**增加变化，实现成本低、单次决策短、带一点记忆点。

---

## 一、陷阱类小玩法

| 名称 | 效果（一句话） | 趣味点 | 实现要点 |
|------|----------------|--------|----------|
| **幻阵** | 本层 3 格暂时打乱顺序（不显示类型，只显示「甲/乙/丙」），再点一次才揭晓 | 开完一个后「剩下两个谁是谁」的猜测 | 本层加状态：格子显示甲乙丙，点甲/乙/丙再按原逻辑揭示 |
| **蚀灵陷阱** | 扣当前生命的 15%（非固定数值） | 血多时痛、血少时轻，决策感不同 | 伤害 = max(5, floor(playerHp * 0.15)) |
| **二选一陷阱** | 「受 20 点伤害」或「失去本层已获得的 1 个机缘效果」 | 要血还是要 buff 的取舍 | 弹小窗两个按钮，选一种执行 |
| **延迟陷阱** | 不立刻扣血，下一层开始时扣 10 点（可叠多层） | 「先开完这层再说」的拖延与风险 | 挂一个 debuff 列表，下一层开始时结算并清空该层挂的 |
| **诅咒格** | 本局接下来 3 次「机缘」格无效（不给你 buff） | 开到一个诅咒后，再开机缘会心塞 | 计数器 curseBlockBuff = 3，机缘生效时 if (curseBlockBuff>0) 不减 buff 但 curseBlockBuff-- |
| **毒瘴** | 每层开始时扣 3 点血，持续 3 层（可叠） | 连开两个毒会压力拉满 | debuff: { type:'poison', layersLeft: 3, dmgPerLayer: 3 }，goNextLayer 时结算 |
| **试炼陷阱** | 答一道极简题（如「3+5=？」选 6/8/9）答对免伤，答错扣 15 | 打破纯随机，有一点「脑子救你一命」 | 弹窗 1 题 3 选项，对则 return，错则扣血 |
| **赌命** | 50% 扣 25 血，50% 回复 15 血 | 明确赌一把，情绪起伏大 | random() < 0.5 扣 else 加，飘字 "+15" 或 "-25" |

## 二、机缘类小玩法

| 名称 | 效果（一句话） | 趣味点 | 实现要点 |
|------|----------------|--------|----------|
| **选择机缘** | 三选一：+10% 攻 / +20 最大生命 / 本层已开格再开一次（仅本层） | 每局不同选择，有 build 感 | 弹窗 3 个按钮，写死 3 种效果 |
| **下次陷阱减半** | 本局下一次陷阱伤害减半（只生效一次） | 开陷阱时「我有盾」的安心感 | flag nextTrapHalf = true，陷阱扣血时 if (nextTrapHalf) { dmg/=2; nextTrapHalf=false } |
| **窥视** | 随机揭示本层另外一格的类型（不打开，只显示文字） | 信息差：知道一个「？」是啥再决定开不开 | 随机选一个未揭示格，临时显示其类型 2 秒或常显到本层结束 |
| **双生机缘** | 本格给一个小 buff，但下一格若为陷阱则陷阱伤害 +50% | 收益与风险绑定 | 挂 debuff：nextTrapBonus = 1.5；陷阱结算时乘算后清掉 |
| **许愿** | 从「下一层 3 格中必有一个灵植」里二选一：灵植 / 灵石 | 轻度控制下一层，有「心想事成」感 | 下一层生成时若本层开过许愿，强制 1 格为选定类型 |
| **存一层** | 本层不拿任何东西，下一层 3 格奖励全部 +50%（灵植/灵石/天材地宝数值乘 1.5） | 忍一层换下一层爆发 | 挂 buff：nextLayerBonus = 1.5，下一层 resolve 收益时乘算 |
| **机缘·窥探** | 随机把本层某一格变成「已揭示但未打开」（只看类型，不拿奖励/不触发陷阱） | 纯信息，再决定开哪两格 | 随机一格 revealed=true 且不触发 resolve，只刷新 UI |
| **机缘·换位** | 本层 3 格类型随机两两交换一次 | 以为要开陷阱结果变灵植的惊喜 | 生成层后 shuffle 一次 3 格的 type |

## 三、组合型（陷阱+机缘一起玩）

| 名称 | 效果 | 趣味点 | 实现要点 |
|------|------|--------|----------|
| **祸福格** | 先随机判定：50% 按陷阱算，50% 按机缘算（再随机一种机缘） | 名字和结果的反差 | 打开时 random() 决定走陷阱分支还是机缘分支 |
| **代价机缘** | 给你一个强力 buff（如 +25% 攻），但本层必须再开 1 格（且若该格是陷阱，伤害 x1.5） | 要 buff 就要多开一格承担风险 | 给 buff + 设 mustOpenOneMore = true，本层再开一格后清掉；陷阱伤害乘 1.5 |
| **陷阱·转机** | 扣 10 血，但本局下一个「机缘」格效果 x2 | 用血换后续机缘质量 | 扣血 + 设 nextBuffDouble = true，机缘生效时乘 2 并清掉 |

## 四、优先可做的 3 个（实现量小）

1. **赌命**：陷阱子类型，50% 扣 25 / 50% 回 15，飘字区分，情绪强。
2. **选择机缘**：机缘子类型，弹窗三选一（+10% 攻 / +20 最大生命 / 下次陷阱减半），立刻生效。
3. **下次陷阱减半**：单独一个小机缘，一个布尔 flag，陷阱结算时判断并减半一次。

这三个都不改大流程，只加分支和 1～2 个状态，就能明显多一层「小玩法」感。
