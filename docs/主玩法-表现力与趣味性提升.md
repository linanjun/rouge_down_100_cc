# 主玩法：表现力与趣味性提升

## 一、主玩法是什么（当前锚定）

- **局外**：洞府养成（境界/修为/灵石）→ 提升生命与攻击，为局内提供数值基础。
- **局内**：**搜**（节点推进）→ **打**（遇敌战斗）→ **撤**（随时撤出或到达出口结算）。

**核心决策**目前只有三处：
1. 局外：是否修炼突破（修为够就突破，无成本取舍）。
2. 局内·搜：当前节点下「前进」还是「撤出」（已知下一节点类型时，决策仍偏直觉）。
3. 局内·打：何时点「符咒」（一次高伤，无 CD 外成本）。

**核心体验目标**：修仙题材下的「探险—战斗—见好就收」的节奏感，以及「变强—再探更深」的成长感。

---

## 二、表现力（玩家能「看到、感受到」什么）

表现力 = 玩法通过反馈、节奏、氛围让玩家**感受到紧张、成长、取舍、修仙感**的程度。

### 当前不足

| 维度 | 现状 | 问题 |
|------|------|------|
| **战斗反馈** | 普攻/符咒只有数值扣血，敌人无受击、无血条、无死亡表现 | 打中/打死缺乏「打到了」的爽感 |
| **节奏与张力** | 节点固定顺序，无「前面可能是危险」的未知感；战斗无阶段变化 | 搜与打都偏平，缺少起伏 |
| **成长可见** | 境界只改数值，局内无「这次比上次强」的对比感 | 养成反馈弱 |
| **修仙氛围** | 界面偏通用 UI，无符纹、灵气、境界意象 | 题材感不足 |
| **撤的权重** | 撤出与到达出口只差文案和少量修为，风险/收益不清晰 | 「撤」的抉择感不强 |

### 提升方向（按性价比排序）

1. **战斗表现（高性价比）**
   - 敌人**血条**（或血条条数）：让「打中」和「快死了」可见。
   - **受击反馈**：受击时骨骼闪白/后仰/顿帧 0.05s，死亡时简短消散或倒地。
   - **攻击表现**：普攻/符咒有**攻击前摇 + 判定帧**，符咒有范围圈或弹道轨迹。
   - 效果：立刻提升「打」的表现力，不改变核心逻辑。

2. **节点与搜的张力**
   - **未知感**：下一节点不直接写死类型，用「？/灵雾/煞气」等模糊描述，前进后才揭晓（战斗/灵药/陷阱/出口）。
   - **路径分支**：当前节点 2～3 条分支（如「左路/右路/直行」），不同分支难度与收益不同。
   - **深度与风险**：越深（步数/层数）怪物越强、收益越高，撤出时「带多少走」的紧张感更强。
   - 效果：让「搜」有信息差和取舍，而不是纯线性点前进。

3. **撤的抉择感**
   - 撤出时**结算界面**明确对比：「若再走一步可能 +X 灵石但可能遇到强敌」。
   - 或加入「撤出需 1 步冷静期」：点撤出后下一节点才是「撤出中」，若该节点是战斗则被迫打一场再撤。
   - 效果：让「撤」成为有代价、有信息的决策，而不是无脑保底。

4. **修仙氛围**
   - 洞府：境界用「炼气/筑基/金丹」等命名，突破时简短**突破演出**（光效/一字提示）。
   - 战斗：符咒有名称 + 简短符纹或范围圈；敌人用「妖兽/邪修」等命名并配 1 句台词或状态描述。
   - UI：主色与形状向「墨/纸/符」靠拢（即使仍是几何，也更有符号感）。
   - 效果：在不改玩法的情况下，强化题材与世界观。

5. **成长可见**
   - 局内可对比：进入战斗时短暂显示「境界 X 层」或「本次秘境第 X 战」。
   - 敌人强度与名字随境界或深度变化（如「炼气妖兽 / 筑基妖兽」）。
   - 效果：让「洞府养成」在局内有可感知的差异。

## 三、趣味性（玩家为什么愿意「再玩一局」）

趣味性 = **决策有深度、操作有手感、结果有差异、目标有追求**。

### 当前不足

| 维度 | 现状 | 问题 |
|
|------|------|------|
| **决策深度** | 节点顺序固定；撤出与继续的收益差不大且不直观 | 选择很快变成「要么冲到底要么随便撤」 |
| **操作与技巧** | 战斗只能移动（若有）+ 点符咒，无走位、无打断、无资源管理 | 技巧空间小 |
| **局内差异** | 每局路径相同，敌人种类与行为单一 | 重复感强 |
| **目标与追求** | 只有境界+数值，无「这次我想拿到某物/达成某条件」 | 单局目标弱 |

### 提升方向（按性价比排序）

1. **让「搜」有决策**
   - **随机路径**：每局节点类型/顺序/分支随机（用种子或固定表保证可平衡）。
   - **二选一/三选一**：每步在 2～3 个「？」中选一个前进，不同选项对应不同风险与收益标签（如「灵药/妖兽/机缘」）。
   - **事件选项**：部分节点是事件（如「发现洞窟：深入 / 绕行」），选错可能受伤或浪费步数。
   - 效果：每局路线不同，重复感下降，决策有意义。

2. **让「打」有操作空间**
   - **移动**：玩家可拖动或摇杆移动，通过走位拉怪、躲范围、等 CD。
   - **技能成本**：符咒消耗「灵力」或「本局次数」，需要权衡「这波用不用」。
   - **敌人差异**：至少 2～3 种（近战冲脸 / 远程 / 高血慢速），对应不同走位与集火顺序。
   - 效果：从「站桩+偶尔点符咒」变成「走位+择机放符咒」，趣味与技巧提升。

3. **局内成长与取舍**
   - **秘境内的临时成长**：如「本局获得符咒·雷」在当次秘境可用，或「本局 +10% 攻击」的 buff 节点。
   - **取舍**：例如「选灵石还是选灵药」「选回复还是选下次战斗伤害+」。
   - 效果：单局内有 build 感和「这次我想试一套打法」的目标。

4. **局外目标与长期追求**
   - **洞府建筑**：丹房（炼药）、藏经阁（解锁符咒）、灵田（产出基础资源），用灵石/材料解锁或升级。
   - **符咒/功法**：局外解锁多种符咒或被动，局内只能带 1～2 个或随机到其中几个。
   - **成就/挑战**：如「无伤通过一层」「单局收集 X 灵药」给予额外奖励或称号。
   - 效果：让「再玩一局」有明确动机（冲境界、解锁新符咒、完成挑战）。

5. **风险与收益的清晰对比**
   - 节点上显示**预期收益区间**（如「可能 5～15 灵石 / 可能一战」），撤出时显示「已获得 vs 再进一步预期」。
   - 死亡惩罚明确（如「本局收获减半」或「无惩罚但无出口奖励」），让「冲还是撤」有算账感。
   - 效果：决策从「感觉」变成「可算」，高玩与普通玩家都能找到乐趣。

## 四、实施优先级建议

| 优先级 | 内容 | 目标 | 实现难度 |
|--------|------|------|----------|
| P0 | 敌人血条 + 受击/死亡表现 + 符咒范围或轨迹 | 战斗「打得爽、看得懂」 | 低 |
| P0 | 节点随机/分支 + 下一节点「？」未知 | 搜有悬念与决策 | 中 |
| P1 | 玩家可移动 + 符咒有资源或次数限制 | 打有操作与取舍 | 中 |
| P1 | 2～3 种敌人类型 + 不同行为 | 战斗有变化 | 中 |
| P2 | 撤出时的收益对比与「再一步」提示 | 撤的抉择感 | 低 |
| P2 | 境界命名 + 突破/符咒的简短演出 | 修仙表现力 | 低 |
| P3 | 局内临时符咒/buff + 洞府建筑 | 长期趣味与目标 | 高 |

---

## 五、总结

- **表现力**：先做「战斗反馈（血条、受击、死亡、技能表现）」和「搜的未知与分支」，再做氛围与成长可见度。
- **趣味性**：先做「搜的决策（随机/分支/？）」和「打的维度（移动、技能成本、敌人种类）」，再做局内 build 与局外长期目标。

主玩法保持「洞府养成 + 搜打撤」不变，通过**反馈更清晰、决策更有信息、操作更有空间、单局更有差异**，把同一套循环做出层次感和重复可玩性。
