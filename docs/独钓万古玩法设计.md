# 独钓万古玩法设计

## 定位

独钓万古是一个局外轻养成玩法，核心体验不是直接打数值，而是“绑定别人的机缘，等待对方破境，再吃到一口反哺”。

它承担三件事：

- 给中后期玩家一个不进秘境也能点一下的长期养成入口。
- 消耗多余灵石，形成新的局外资源去向。
- 用“借他人因果修自己道果”的方式，补足修仙世界的故事感和关系感。

## 核心循环

1. 玩家进入独钓万古界面。
2. 购买机缘鱼饵，升级方杆。
3. 根据自身境界开放的绑定位数量，选择要观察的机缘修士。
4. 点击抛竿，消耗 1 份鱼饵。
5. 若某个绑定对象触发机缘并提升境界，则玩家获得一部分修为反哺。
6. 若当前绑定对象收益一般，可直接替换为其他修士。

## 资源设计

### 机缘鱼饵

- 用途：每次抛竿消耗 1 份。
- 产出：局外购买。
- 作用：没有鱼饵时不能触发独钓万古玩法。

### 方杆

- 用途：独钓专属钓具等级。
- 产出：局外用灵石升级。
- 作用：提升机缘触发率，并略微放大反哺收益。

## 绑定位设计

- 初始开放 1 个位置。
- 随自身境界提升逐步开放更多位置。
- 建议节奏：
  - 练气：1 位
  - 筑基：2 位
  - 金丹：3 位
  - 元婴：4 位
- 每个位都可以绑定一个机缘修士。
- 绑定后允许随时替换，不需要额外解锁替换功能。

## 机缘修士设计

每个修士拥有以下属性：

- 名称
- 当前境界
- 反哺比例
- 风格说明

风格示例：

- 稳定型：容易触发，但单次回馈偏平。
- 猛进型：触发少，但一旦破境回馈更高。
- 后期型：前期收益一般，后期境界抬高后价值明显。

## 反哺规则

- 玩家抛竿后，对每个已绑定修士分别判定一次是否触发机缘。
- 触发后该修士提升 1 到 2 层境界。
- 系统根据该修士提升后的境界，按其反哺比例返还一部分修为给玩家。
- 若本次没有任何修士触发机缘，仍给少量保底修为，避免手感过空。

## 体验目标

- 点一下就能获得反馈，不要做成过长的等待玩法。
- 每个修士都要有“这人值不值得绑”的差异感。
- 替换绑定要足够顺滑，降低试错成本。
- 数值不抢主玩法，但要成为稳定补充。

## 后续可扩展方向

- 真玩家因果池：把假想修士替换为服务器中的真实玩家镜像。
- 因果词条：不同修士额外携带“暴击反哺”“双倍触发”“延迟大回馈”等标签。
- 钓台事件：抛竿时概率钓上古遗物、残卷、秘闻或特殊追踪对象。
- 社交化：允许关注某个好友，优先出现在因果池中。
