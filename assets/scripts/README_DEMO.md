# 像素武侠修仙角色界面 — 扁平化设计

## 设计特点
- **扁平化**: 无圆角、无渐变、无阴影
- **纯色方案**: 所有颜色均为纯色，通过 Graphics 矩形绘制
- **边框实现**: 外层矩形（边框色）+ 内层矩形（背景色）叠加

## 运行方式
- 用 Cocos Creator 3.8 打开工程，运行任意**带 Canvas 节点的场景**即可。
- 场景加载后，`FrameworkPage.ts` 会自动挂到 `Canvas` 下创建 `FrameworkRoot`（720×1280 框架壳）。
- 随后 `CharacterPage.ts` 会自动挂到 `FrameworkRoot/Content` 节点下填充内容区。

## 当前架构
- **FrameworkPage.ts** — 720×1280 通用框架壳（扁平化）
  - 顶部栏 (84px)：头像 | 名字 + 境界 | 金币 | 钻石
  - 内容区：可由外部填充
  - 功能导航栏 (100px)：任务 | 商店 | 炼丹 | 炼器 | 功法 | 灵宠
  - 底部标签栏 (90px)：本命法器(激活) | 人物 | 境界 | 洞天 | 钓鱼

- **CharacterPage.ts** — 角色总览页（扁平化）
  - 主内容区 (~360px)：
    - 标题横幅（角色总览，蓝底金边）
    - 本命法器面板（飞剑位/护符位/灵灯位，各 Lv.1）
    - 角色形象面板（修炼台座 + 星光装饰）
  - 底部面板组 (~240px)：
    - 基础属性面板（气血 160/160 / 法力 88/88 / 行动力 180/180）
    - 功法造诣面板（炼丹术/炼器术/灵宠诀/清心诀，各 Lv.1 0/13）

## 颜色规范
- 背景：slate 系列（#1e293b 主背景）
- 强调：amber 系列（金黄色边框和标题）
- 功法：blue 系列（蓝色面板）
- 属性：red/blue/green（气血/法力/行动力）
- 钻石：cyan 系列
