# 像素武侠修仙角色界面 说明

## 运行方式
- 用 Cocos Creator 3.8 打开工程，运行任意**带 Canvas 节点的场景**即可。
- 场景加载后，`FrameworkPage.ts` 会自动挂到 `Canvas` 下创建 `FrameworkRoot`（720×1280 框架壳）。
- 随后 `CharacterPage.ts` 会自动挂到 `FrameworkRoot/Content` 节点下（696×940 内容区）。

## 当前架构
- **FrameworkPage.ts** — 720×1280 通用框架壳
  - 顶部栏：头像 | 名字 + 境界 | 金币 | 钻石
  - 内容区：可由外部填充（696×940）
  - 活动栏：任务 | 商城 | 炼丹 | 炼器 | 功法 | 灵宠
  - 底部栏：法器 | 角色 | 秘境 | 洞天 | 独钓

- **CharacterPage.ts** — 角色总览页
  - 标题横幅（角色总览）
  - 本命法器面板（飞剑位 / 护符位 / 灵灯位）
  - 角色形象面板（修炼台座 + 角色立绘区）
  - 基础属性面板（气血 / 法力 / 行动力进度条）
  - 功法造诣面板（炼丹术 / 炼器术 / 灵宠诀 / 清心诀）
