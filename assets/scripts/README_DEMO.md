# 洞府·搜打撤 Demo 说明

## 运行方式
- 用 Cocos Creator 3.8 打开工程，运行任意**带 Canvas 节点的场景**即可。
- 脚本 `GrottoExpeditionDemo.ts` 会在场景加载后自动挂到 `Canvas` 下（创建 `DemoRoot` 节点），无需在编辑器里手动挂组件。

## 玩法流程
1. **洞府（局外）**  
   - 查看境界、灵石、修为。  
   - 修为满可点「修炼突破」提升境界（加生命、攻击）。  
   - 点「进入秘境」开始一局探索。

2. **秘境（局内·搜）**  
   - 按节点推进：当前节点显示名称与说明。  
   - **前进**：进入下一节点。若为「妖兽拦路」则进入战斗；灵药/灵雾则获得收益；出口则结算并返回洞府。  
   - **撤出秘境**：随时带着当前收获离开并结算。

3. **战斗（局内·打）**  
   - 己方角色与敌人均为**骨骼动画**（程序化肢体：身体、头、双臂、双腿），自动播放待机/攻击/奔跑。  
   - 敌人会追向玩家并近战扣血；玩家自动对范围内敌人进行普攻，可点击「符咒」释放一次范围伤害。  
   - 全灭敌人后自动回到秘境节点并继续前进。

4. **结算**  
   - 撤出或到达出口后，显示本局灵石与修为，点「返回洞府」回到局外。

## 骨骼动画
- 角色与敌人使用同一套 `CharacterRig`：root → body → head，body 下 armL/armR，root 下 legL/legR。  
- 每段肢体为 `Graphics` 绘制的圆角矩形，父子层级控制朝向与位置。  
- 玩家：待机轻微摆动 + 攻击时手臂前挥。  
- 敌人：奔跑时四肢摆动。
