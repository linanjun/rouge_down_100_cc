# 像素武侠修仙角色界面 — 扁平化设计

## 设计特点
- **扁平化**: 无圆角、无渐变、无阴影
- **纯色方案**: 所有颜色均为纯色，通过 Graphics 矩形绘制
- **边框实现**: 外层矩形（边框色）+ 内层矩形（背景色）叠加

## 运行方式
- 用 Cocos Creator 3.8 打开工程，运行任意**带 Canvas 节点的场景**即可。
- 场景加载后，`FrameworkPage.ts` 会自动挂到 `Canvas` 下创建 `FrameworkRoot`（720×1280 框架壳）。
- 随后 `CharacterPage.ts` 会自动挂到 `FrameworkRoot/Content` 节点下填充内容区。

## 当前架构
- **FrameworkPage.ts** — 720×1280 通用框架壳（扁平化）
  - TopBar 顶部栏 (106px)：头像 | 名字+境界 | 金币+钻石
  - NavigationBar 功能导航栏 (80px)：任务 | 商店 | 炼丹 | 炼器 | 功法 | 灵宠
  - Content Area 内容区：可由外部填充
  - BottomNavBar 底部标签栏 (80px)：法器 | 独钓 | 角色(激活) | 秘境 | 洞天

- **CharacterPage.ts** — 角色页面（扁平化）
  - 角色形象展示区：
    - 标题横幅（角色总览，蓝底金边）
    - 角色形象 + 修炼台座 + 星光装饰
    - 4个装备槽位（武器/防具/饰品/法宝，四角分布）
  - 属性面板：
    - 战力 12580 / 等级 Lv.10 / 经验进度条
    - 攻击 850 / 防御 620 / 速度 75 / 暴击 45%

## 颜色规范
- 背景：slate 系列（#1e293b 主背景）
- 强调：amber 系列（金黄色边框和标题）
- 功法：blue 系列（蓝色标题横幅）
- 钻石：cyan 系列
